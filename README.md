Esta aplicação suporta a versão 3.8.5 do Python

Para Instalar:

````
    Console
    python3 -m venv .venv 
    source .venv/bin/activate
    pip install -r requirements-dev.txt
````

Para conferir a qualidade do código:
 ```
    flaske8
```
